# 资源文件说明

## 图标文件

本项目的图标使用 SVG 格式，请在此目录下放置以下图标文件：

### 必需的图标文件：
- `add.svg` - 添加任务图标
- `check.svg` - 完成图标
- `delete.svg` - 删除图标
- `edit.svg` - 编辑图标
- `priority_high.svg` - 高优先级图标
- `priority_medium.svg` - 中优先级图标
- `priority_low.svg` - 低优先级图标

### 图标设计建议：
- 使用 24x24 的 viewBox
- 使用简洁的线条设计
- 颜色建议：
  - 主色调：#3B82F6 (蓝色)
  - 成功色：#10B981 (绿色)
  - 警告色：#F59E0B (橙色)
  - 危险色：#EF4444 (红色)

### 图标获取途径：
1. 使用 Heroicons (https://heroicons.com/)
2. 使用 Lucide Icons (https://lucide.dev/)
3. 自定义设计

## 样式文件

样式文件由 `ThemeManager` 类动态生成，不需要额外的样式文件。
